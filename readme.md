MelodyHub
Descrição do Projeto
O MelodyHub é um aplicativo de streaming de música inspirado em plataformas como o Spotify. Este projeto acadêmico tem como foco o aprendizado de arquitetura de software, desenvolvimento backend e boas práticas de programação.

O sistema permite que usuários criem contas, gerenciem playlists, adicionem músicas favoritas e realizem assinaturas de planos pagos. Além disso, o projeto implementa regras de antifraude em transações financeiras, garantindo segurança e consistência no uso do serviço.

A API do MelodyHub deve ser construída utilizando Spring Boot, seguindo os princípios de Domain Driven Design (DDD), SOLID, Clean Code e padrões de repositórios e serviços de domínio, promovendo a construção de domínios ricos e sustentáveis.

Objetivo do Projeto
O objetivo principal é desenvolver uma API robusta e escalável, que atenda às operações típicas de um serviço de streaming de música e implemente regras de negócio e antifraude de forma modular, clara e extensível.

Especificamente, os alunos devem:

Modelar domínios complexos e agregados usando DDD;
Aplicar os princípios SOLID e Clean Code na implementação;
Desenvolver interfaces e APIs REST com intenção reveladora;
Garantir integridade e segurança de transações financeiras;
Praticar boa arquitetura, incluindo Context Mapping, Domain Services e Repository Pattern
